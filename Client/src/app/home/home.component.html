Mars Rover NASA API Photo Project

Dit is een project gemaakt door Maxim Wauters voor het vak Cloud APIs. In dit project heb ik gebruik gemaakt van de NASA API om bepaalde gegevens op te vragen. Ook heb ik een eigen REST API gemaakt voor foto's die je leuk vind in te bewaren. U kan makkelijk door de site navigeren via het menu.

In de rubriek Gallery worden via paging enkele foto's die de Mars Rover genomen heeft afgebeeld. U kan vervolgens ook foto's opzoeken op datum...

In de rubriek My Photo List kan je foto's die je leuk vind bewaren aan de hand van een Titel, description en Url van de foto.
